| 検索                   | コマンド                     |
| ---------------------- | ---------------------------- |
| 完全一致検索           | “A”                          |
| 除外                   | -A                           |
| or検索                 | A OR B                       |
| OR+AND検索             | (A OR B) C                   |
| 関連検索               | related:URL A                |
| 特定サイト内検索       | site:URL A                   |
| キャッシュ検索         | cache:URL A                  |
| ファイル検索           | filetype:拡張子 A            |
| 期間指定               | 検索結果→ツール→期間指定なし |
| 全てのキーワードを含む | allintext:A B C              |
